# 安知鱼主题接入51LA统计展示访问数据

## 前言

安知鱼主题的关于页自带了一个统计卡片，可以展示<span style="color: rgba(53, 163, 241, 1)">今日访客、今日浏览、昨日访客、昨日浏览、本月浏览、总浏览量</span>等等这些数据，支持51LA和Umami两种统计来源。这两个基本都是免费的，先来看看效果

## 效果预览

配置完成后，关于页会多出一个卡片：
![51LA预览](https://yuumii.top/article/51LA/51LA%E7%BB%9F%E8%AE%A1.png)

## 实现步骤

### 第一步：注册 51LA 并创建应用

1. 打开 [v6.51.la](https://v6.51.la/)，注册一个账号（免费的，直接注册就行）
2. 登录后进入 **应用管理**，点击 **添加应用**
3. 填写完应用信息之后就可以在这里获取到你的掩码

![51LA掩码获取](https://yuumii.top/article/51LA/51LA%E6%8E%A9%E7%A0%81%E8%8E%B7%E5%8F%96.png)


| 配置项 | 值 | 说明 |
|--------|-----|------|
| 应用名称 | 随便填，比如「我的博客」 |  |
| 网站域名 | 你的博客域名，比如 `yuumii.top` | <span style="color: rgba(247, 79, 79, 1)">必须准确填写</span>，否则统计不到数据 |
| 行业类别 | 随便选一个就行 |  |

4. 添加完成后会得到一段统计代码，按提示粘贴到你的页面中（安知鱼主题会自动加载51LA的统计 SDK，如果已经在用了可以跳过粘贴代码这一步）

### 第二步：开启数据挂件

数据挂件是 51LA 提供的一种前端展示组件，可以把统计数据渲染成一段HTML写入你的页面。

1. 登录 51LA 后台，进入你的应用 → 左侧菜单找到 **数据挂件**
2. 点击 **启用数据挂件功能**

![51LA开启数据挂件](https://yuumii.top/article/51LA/51LA%E5%BC%80%E5%90%AF%E6%95%B0%E6%8D%AE%E6%8C%82%E4%BB%B6.png)


3. 系统会生成一段类似这样的嵌入代码：

```html
<script id="LA-DATA-WIDGET" crossorigin="anonymous" charset="UTF-8"
  src="https://v6-widget.51.la/v6/xxxxxx/quote.js?theme=0&col=true&f=12&display=1,0,0,1,0,1,1,1">
</script>
```

<span style="color: rgba(247, 79, 79, 1)">这里要稍微等个几分钟 然后才能够正常使用这个接口 这个ID其实就是安知鱼配置里面的CK 复制进去即可


### 第三步：修改主题配置

打开安知鱼主题的配置文件 `_config.anzhiyu.yml`，找到 51LA 统计配置：

```yaml
# 51la统计配置
LA:
  enable: true                     # 开启统计卡片
  ck: "3QMSDLh40WIi9Wr4"          # 在应用页面有  数据挂件页面最下面的链接里面也有
  LingQueMonitorID:                # 灵雀监控ID，一般不用填
```

| 配置项 | 说明 |
|--------|------|
| `enable` | 统计卡片总开关，`true` 开启，`false` 关闭 |
| `ck` | 在应用页面的掩码  数据挂件页面最下面的链接里面也有 |
| `LingQueMonitorID` | 灵雀应用监控的 ID，跟统计没关系，不用填 |


### 第四步：等待并验证

1. 开启数据挂件后，**至少等 10 分钟**
2. 验证挂件是否生效，直接在浏览器中访问你的挂件地址：

```bash
https://v6-widget.51.la/v6/你的ID/quote.js
```

如果返回一段 JavaScript 代码而不是 404，说明挂件已经创建好了。如果还返回 404，再等几分钟刷新试试。

3. 确认挂件生效后，重新构建博客：

```bash
hexo cl && hexo g && hexo s
```

打开博客的关于页，统计卡片应该就能正常显示数据了。
